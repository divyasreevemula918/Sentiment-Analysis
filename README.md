# 🎬 Sentiment Analysis using Deep Learning (Simple RNN)

## 📌 Project Overview

This project builds a **Sentiment Analysis system** that classifies IMDB movie reviews as **Positive 😊 or Negative 😞** using a **Simple RNN model**.

It includes a **live web application** where users can input text and get predictions instantly.

---

## 🌐 Live Demo (Deployment)

👉 Try the app here:
🔗 **https://sentiment-analysis-mne9iwheaydcappcntjmle7.streamlit.app/**

---

## 🚀 Features

* ✅ Predict sentiment of movie reviews
* ✅ Deep Learning model using Simple RNN
* ✅ Real-time prediction via web app
* ✅ Interactive Streamlit UI
* ✅ Text preprocessing and sequence padding

## ⚙️ How It Works

1. User enters a movie review
2. Text is preprocessed (tokenized, padded)
3. Model predicts sentiment
4. Output shown as Positive or Negative
